Registration Form Assignment

Assignment Instructions

  1. Open the assignment file: Open assignments/registration.json to see the form requirements
  2. Study the requirements: The JSON file contains all field specifications including:
    • Field IDs and names
    • Field types (text, email, number, password, tel, date, textarea, select, radio, checkbox)
    • Which fields are required
    • Labels and placeholders
    • Validation rules (min, max, pattern, minLength, maxLength)
  3. Create the form: Build a form below with all required fields according to the JSON specification
  4. Requirements:
    • Form must have id="registrationForm"
    • All fields must have correct id and name attributes (as specified in JSON)
    • Use correct input types (text, email, number, password, tel, date, textarea, select, radio, checkbox)
    • Add required attribute to required fields
    • Include correct labels and placeholders from JSON
    • Submit button must have id="submitBtn" and type="submit"
    • Minimum 10 fields required
  5. Implement form submission: Edit script.js to handle form submission:
    • Add event listener for form submit
    • Prevent default form submission behavior
    • Collect form data
    • Display data (console.log, alert, etc.)
  6. Test your solution: Run npm test to verify your form meets all requirements

Example Field Structure:

<div class="form-group">
    <label for="fieldId">Label Text</label>
    <input type="text" id="fieldId" name="fieldId" placeholder="Placeholder text" required>
</div>
